# ImBroker
ImBroker is a message broker written in Go, it has basic functionality for publishing and subscribing to topics. It might be expanded on in the future, but this is all I need for now! <br>
To start the broker, you simply have to start the go program located in server/ <br>
Then you need to connect to localhost:9988, if you want to change the access level, you can change the constants inside the program!
